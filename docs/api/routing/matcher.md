# `routing/matcher`
[`View source`](../../../src/routing/matcher.js)

> Function to match url to given routes

```js
matcher(url, routes);
```

#### Parameters

##### `url : string`

An URL to test against.

##### `routes : Array`

An array of routes to test for.

#### Return value

Matching route if there is positive match.
Otherwise an exception is thrown.
